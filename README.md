## 列表手动更新备忘

- 1. Sublime 开启正则匹配；
- 
- 2. 按F9或者选择菜单： Edit > Sort Lines ，对每行文本进行排序；
- 
- 3. 删除重复行：
> - 排序好后，按 Ctrl+H ，调出替换面板
-
- 4. 查找字符串：
> - `^(.+)$[\r\n](^\1$[\r\n]{0, 1})+`
> - 注意：确保正则模式开关打开；若不可用，按Alt+R进行切换

- 5. 替换字符串：
> - `\1\n`
> - 点击Replace
